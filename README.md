# [bugajski.io](https:bugajski.io)

Made with [Vue.js](https://vuejs.org/) and [Bulma](https://bulma.io/). Assets include [IBM Plex Sans](https://fonts.google.com/specimen/IBM+Plex+Sans), [IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono), and [FontAwesome](https://fontawesome.com/icons/external-link-square-alt?style=solid). Colors are from the [IBM Color Palette](https://www.ibm.com/design/language/color/).

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
